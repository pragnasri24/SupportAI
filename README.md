# 🚀 SupportAI

SupportAI is a full-stack customer support platform that enables users to create, manage, and track support tickets through a modern web interface. The application provides secure authentication, ticket analytics, and role-based dashboards for customers and support agents.

---

## 🌐 Live Demo

### Frontend
https://support-ai-sable.vercel.app

### Backend API
https://supportai-3v3x.onrender.com

---

## ✨ Features

- 🔐 Secure User Authentication (JWT)
- 👤 Customer Registration & Login
- 🎫 Create, View, and Track Support Tickets
- 📊 Ticket Analytics Dashboard
- 🔎 Search and Filter Tickets
- ⚡ Real-time Ticket Updates
- 👨‍💼 Customer & Agent Roles
- ☁️ Cloud Deployment

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/pragnasri24/SupportAI.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

### Backend

Create a `.env` file inside the `server` folder.

```env
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:5173
```
